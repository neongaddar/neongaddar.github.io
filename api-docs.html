API Documentation

Complete reference for all NEONPAY classes, methods, and utilities.

NeonPay Class

The main class for handling Telegram bot payments.

Factory Methods

@classmethod def create_for_aiogram(cls, bot: Bot) -> NeonPay @classmethod def create_for_pyrogram(cls, client: Client) -> NeonPay @classmethod def create_for_ptb(cls, application: Application) -> NeonPay @classmethod def create_for_telebot(cls, bot: TeleBot) -> NeonPay @classmethod def create_for_raw_api(cls, bot_token: str) -> NeonPay

Create a NeonPay instance for your specific bot library.

bot/client/application Bot instance from your library

Your bot instance from the respective library.

Returns

NeonPay: Configured NeonPay instance

# Aiogram example from aiogram import Bot from neonpay import NeonPay bot = Bot("YOUR_TOKEN") neonpay = NeonPay.create_for_aiogram(bot)

PaymentStage Class

Represents a payment stage with product information.

Constructor

PaymentStage( title: str, description: str, price: int, logo_url: Optional[str] = None, callback: Optional[Callable] = None )
title str

Product or service title

description str

Detailed description of the product

price int

Price in Telegram Stars (XTR)

logo_url Optional[str]

URL to product logo/image

callback Optional[Callable]

Function to call when payment is successful

PaymentResult Class

Contains information about a completed payment.

Properties

user_id int

Telegram user ID who made the payment

amount int

Payment amount in Telegram Stars

stage PaymentStage

The payment stage that was purchased

message Message

Original message object from the bot library

payment_id str

Unique payment identifier

def create_stage( self, title: str, description: str, price: int, logo_url: Optional[str] = None, callback: Optional[Callable] = None ) -> PaymentStage

Middleware System

Extend NEONPAY functionality with middleware.

Creating Middleware

from neonpay.middleware import PaymentMiddleware class LoggingMiddleware(PaymentMiddleware): async def before_payment(self, stage, user_id): print(f"Payment starting for user {user_id}") async def after_payment(self, payment_result): print(f"Payment completed: {payment_result.amount} XTR") # Add middleware neonpay.add_middleware(LoggingMiddleware())
