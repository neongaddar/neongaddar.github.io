NeonPay Class

The main class for handling Telegram bot payments.

Класс NeonPay

Основной класс для обработки платежей Telegram ботов.

NeonPay Sinifi

Telegram bot ödənişlərini idarə etmək üçün əsas sinif.

Factory Methods

Фабричные методы

Fabrik metodları

@classmethod def create_for_aiogram(cls, bot: Bot) -> NeonPay @classmethod def create_for_pyrogram(cls, client: Client) -> NeonPay @classmethod def create_for_ptb(cls, application: Application) -> NeonPay @classmethod def create_for_telebot(cls, bot: TeleBot) -> NeonPay @classmethod def create_for_raw_api(cls, bot_token: str) -> NeonPay

Create a NeonPay instance for your specific bot library.

Создайте экземпляр NeonPay для вашей конкретной библиотеки ботов.

Xüsusi bot kitabxananız üçün NeonPay nümunəsi yaradın.

Ваш экземпляр бота из соответствующей библиотеки.

Müvafiq kitabxanadan bot nümunəniz.

# Aiogram example from aiogram import Bot from neonpay import NeonPay bot = Bot("YOUR_TOKEN") neonpay = NeonPay.create_for_aiogram(bot)

payment_handler Decorator

Декоратор payment_handler

payment_handler Dekoratoru

def payment_handler(self, stage: PaymentStage)

Decorator to handle payments for a specific stage.

Декоратор для обработки платежей для конкретного этапа.

Xüsusi bir etap üçün ödənişləri idarə etmək üçün dekorator.

@neonpay.payment_handler(premium_stage) async def handle_premium_payment(payment_result: PaymentResult): # Grant premium access user_id = payment_result.user_id await grant_premium_access(user_id) await payment_result.message.answer( "🎉 Premium access activated!" )

Middleware System

Extend NEONPAY functionality with middleware.

Система middleware

Расширьте функциональность NEONPAY с помощью middleware.

Middleware Sistemi

NEONPAY funksionallığını middleware ilə genişlət.

Creating Middleware

from neonpay.middleware import PaymentMiddleware class LoggingMiddleware(PaymentMiddleware): async def before_payment(self, stage, user_id): print(f"Payment starting for user {user_id}") async def after_payment(self, payment_result): print(f"Payment completed: {payment_result.amount} XTR") # Add middleware neonpay.add_middleware(LoggingMiddleware())
