Our Mission

At NEONPAY, we believe that monetizing Telegram bots should be as simple as creating them. Before NEONPAY, developers had to deal with complex payment integrations, lengthy documentation, and multiple API endpoints just to add a simple payment feature to their bots.


Our mission is to eliminate this complexity and provide a unified, simple API that works across all major Python Telegram bot libraries. We want every developer, regardless of their experience level, to be able to add payments to their bots with just 2-3 lines of code.

Project History

December 2024

NEONPAY 1.0.0 Release

Official stable release with full support for all major Python Telegram bot libraries. Production-ready with comprehensive documentation and examples.

November 2024

Beta Testing Phase

Extensive beta testing with 100+ developers. Feedback collection and bug fixes. Performance optimizations and API refinements.

October 2024

Multi-Library Support

Added support for Pyrogram, python-telegram-bot, and pyTelegramBotAPI. Unified API design across all libraries.

September 2024

Aiogram Integration

Initial release with Aiogram support. Core payment processing functionality and Telegram Stars integration.

August 2024

Project Inception

Project started by Abbas Sultanov after struggling with complex payment integrations in multiple bot projects. Initial research and development phase.
