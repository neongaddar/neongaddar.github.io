Quick Start Guide

Get NEONPAY up and running in your Telegram bot in just a few minutes.

Installation

Install NEONPAY using pip:

pip install neonpay
Python Version: NEONPAY supports Python 3.9 and above.

Basic Setup

NEONPAY works with all major Telegram bot libraries. Here's how to set it up:

1 Import and Initialize
from neonpay import NeonPay from aiogram import Bot bot = Bot("YOUR_BOT_TOKEN") neonpay = NeonPay.create_for_aiogram(bot)
2 Create Payment Stage
stage = neonpay.create_stage( title="Premium Access", description="Unlock all premium features", price=100, # 100 Telegram Stars (XTR) logo_url="https://example.com/logo.png" )

Your First Payment

Create a simple payment handler:

from aiogram import Router from aiogram.types import Message from aiogram.filters import Command router = Router() @router.message(Command("buy")) async def buy_premium(message: Message): # Send payment invoice await neonpay.send_payment(message.chat.id, stage) @neonpay.payment_handler(stage) async def handle_premium_payment(payment_result): # User successfully paid - grant premium access user_id = payment_result.user_id await payment_result.message.answer( "üéâ Welcome to Premium! You now have access to all features." ) # Your logic here: save to database, update user status, etc.

Handling Payments

NEONPAY provides several ways to handle payment results:

Using Decorators

@neonpay.payment_handler(stage) async def handle_payment(payment_result): # Automatic handling for specific stage print(f"Payment received: {payment_result.amount} XTR")

Using Callbacks

async def payment_callback(payment_result): # Custom callback function await send_premium_content(payment_result.user_id) stage = neonpay.create_stage( title="Premium Content", price=50, callback=payment_callback )

Manual Handling

# Listen to all payments @neonpay.on_payment async def handle_any_payment(payment_result): stage_title = payment_result.stage.title if stage_title == "Premium Access": # Handle premium access pass elif stage_title == "Digital Product": # Handle digital product pass
