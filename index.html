NEONPAY API

Simple and elegant Telegram Stars payment integration for Python bots. Add payments to your bot with just 2-3 lines of code.

NEONPAY API

Простая и элегантная интеграция платежей Telegram Stars для Python ботов. Добавьте платежи в ваш бот всего 2-3 строками кода.

NEONPAY API

Python botları üçün sadə və zərif Telegram Stars ödəniş inteqrasiyası. Cəmi 2-3 kod sətri ilə botunuza ödənişlər əlavə edin.

Quick Example

Быстрый пример

Sürətli nümunə

# Install NEONPAY pip install neonpay # Simple payment example from neonpay import NeonPay from aiogram import Bot bot = Bot("YOUR_BOT_TOKEN") neonpay = NeonPay.create_for_aiogram(bot) # Create payment stage stage = neonpay.create_stage( title="Premium Access", description="Unlock all premium features", price=100, # 100 XTR (Telegram Stars) logo_url="https://example.com/logo.png" ) # Handle payment @neonpay.payment_handler(stage) async def handle_premium_payment(payment_result): await payment_result.message.answer("Welcome to Premium! 🎉")

Key Features

Easy Integration

Add payments to your bot in just 2-3 lines of code. No complex setup required.

Multi-Library Support

Works with Aiogram, Pyrogram, python-telegram-bot, pyTelegramBotAPI, and raw API.

Telegram Stars

Native support for Telegram Stars (XTR) payments with automatic handling.

Secure

Built-in validation and error handling for secure payment processing.
