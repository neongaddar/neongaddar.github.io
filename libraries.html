Installation & Setup

Установка и настройка

Quraşdırma və tənzimləmə

Basic Installation

Install NEONPAY with basic functionality:

Базовая установка

Установите NEONPAY с базовой функциональностью:

Əsas quraşdırma

NEONPAY-ı əsas funksionallıqla quraşdırın:

pip install neonpay

With Aiogram

Install with Aiogram support:

С Aiogram

Установите с поддержкой Aiogram:

Aiogram ilə

Aiogram dəstəyi ilə quraşdırın:

pip install neonpay[aiogram]

With Pyrogram

Install with Pyrogram support:

С Pyrogram

Установите с поддержкой Pyrogram:

Pyrogram ilə

Pyrogram dəstəyi ilə quraşdırın:

pip install neonpay[pyrogram]

With python-telegram-bot

Install with PTB support:

С python-telegram-bot

Установите с поддержкой PTB:

python-telegram-bot ilə

PTB dəstəyi ilə quraşdırın:

pip install neonpay[ptb]

With pyTelegramBotAPI

Install with telebot support:

С pyTelegramBotAPI

Установите с поддержкой telebot:

pyTelegramBotAPI ilə

telebot dəstəyi ilə quraşdırın:

pip install neonpay[telebot]

All Libraries

Install with support for all libraries:

Все библиотеки

Установите с поддержкой всех библиотек:

Bütün kitabxanalar

Bütün kitabxanalar üçün dəstəklə quraşdırın:

pip install neonpay[all]
