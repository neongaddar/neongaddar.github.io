Supported Libraries

NEONPAY works seamlessly with all major Telegram bot libraries through our adapter system.

Aiogram v3
Modern async Telegram Bot API framework
  • Full async/await support
  • Type hints and modern Python features
  • Built-in middleware system
  • Webhook and polling support
from aiogram import Bot from neonpay import NeonPay bot = Bot("TOKEN") neonpay = NeonPay.create_for_aiogram(bot) stage = neonpay.create_stage( title="Premium", price=100 ) @neonpay.payment_handler(stage) async def handle_payment(result): # Handle successful payment pass
Pyrogram v2+
Elegant, modern Telegram MTProto API framework
  • MTProto and Bot API support
  • Clean and intuitive API
  • Excellent documentation
  • User and bot account support
from pyrogram import Client from neonpay import NeonPay app = Client("bot", bot_token="TOKEN") neonpay = NeonPay.create_for_pyrogram(app) stage = neonpay.create_stage( title="Premium", price=100 ) @neonpay.payment_handler(stage) async def handle_payment(result): # Handle successful payment pass
python-telegram-bot
Pure Python Telegram Bot API wrapper
  • Comprehensive Bot API coverage
  • Excellent error handling
  • Rich ecosystem of extensions
  • Well-established and stable
from telegram.ext import Application from neonpay import NeonPay app = Application.builder().token("TOKEN").build() neonpay = NeonPay.create_for_ptb(app) stage = neonpay.create_stage( title="Premium", price=100 ) @neonpay.payment_handler(stage) async def handle_payment(result): # Handle successful payment pass
pyTelegramBotAPI
Simple and lightweight Telegram Bot API wrapper
  • Simple and easy to use
  • Sync and async support
  • Minimal dependencies
  • Great for beginners
import telebot from neonpay import NeonPay bot = telebot.TeleBot("TOKEN") neonpay = NeonPay.create_for_telebot(bot) stage = neonpay.create_stage( title="Premium", price=100 ) @neonpay.payment_handler(stage) async def handle_payment(result): # Handle successful payment pass
Raw Telegram Bot API
Direct HTTP requests to Telegram Bot API
  • No external dependencies
  • Full control over requests
  • Webhook support included
  • Perfect for microservices
from neonpay import NeonPay neonpay = NeonPay.create_for_raw_api("TOKEN") stage = neonpay.create_stage( title="Premium", price=100 ) @neonpay.payment_handler(stage) async def handle_payment(result): # Handle successful payment pass
Auto-Detection
Automatic library detection and adapter creation
  • Automatically detects your bot library
  • No need to specify adapter type
  • Works with any supported library
  • Simplifies integration process
from neonpay import NeonPay # Works with any bot instance neonpay = NeonPay.create_auto(your_bot_instance) # Or detect from context neonpay = NeonPay.auto_detect() stage = neonpay.create_stage( title="Premium", price=100 )

Installation & Setup

Basic Installation

Install NEONPAY with basic functionality:

pip install neonpay

With Aiogram

Install with Aiogram support:

pip install neonpay[aiogram]

With Pyrogram

Install with Pyrogram support:

pip install neonpay[pyrogram]

With python-telegram-bot

Install with PTB support:

pip install neonpay[ptb]

With pyTelegramBotAPI

Install with telebot support:

pip install neonpay[telebot]

All Libraries

Install with support for all libraries:

pip install neonpay[all]
